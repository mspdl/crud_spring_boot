## CRUD em Spring Boot

Projeto criado a partir do Curso # Modelagem de Dados UML Análise&Projeto Orientado a Objetos) do Udemy.
O projeto foi desenvolvido utilizando as seguintes tecnologias:
 - Java 12
 - Framework Spring Boot
 - Banco de dados H2
 - Arquitetura MVC
 - Web services rest

O projeto possui três endpoints:
 - /pedidos/{id}
 -  /clientes/{id]
 - /categorias/{id}

Já existem dados pré-cadastrados para testar os end-points e a aplicação em si, esses dados estão configurado no arquivo CursomcApplication.java
